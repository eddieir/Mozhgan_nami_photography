# Mozhgan Nami Photography

Fine art photography website built with React + Vite + Framer Motion.

## Stack
- React 18 + Vite
- Framer Motion (animations)
- Lucide React (icons)
- Pure CSS design system (no Tailwind)

## Dev
```bash
npm install
npm run dev
```

## Deploy to Netlify
1. Push to GitHub: `git push origin main`
2. Connect repo in Netlify dashboard
3. Build command: `npm run build`  
4. Publish directory: `dist`

Or drag-and-drop the `dist/` folder to Netlify.
